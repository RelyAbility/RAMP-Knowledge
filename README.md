## Architecture Philosophy

**RAMP Knowledge Repository** (This repo)
- Conceptual + Strategic knowledge
- Industry intelligence deep-dives
- Validation frameworks and methodologies
- Long-term ecosystem architecture
- **Single source of truth** for all RAMP modules

**GitHub Operational Repositories** (e.g., ESA-V3-02-03-2026-Latest)
- Executable code + reference data
- Operational procedures (how to run, validate, integrate)
- Version-controlled deliverables
- PR discipline and schema enforcement
- **Reference back to RAMP** for context

**Prevents Documentation Drift**: Clear boundary between strategic knowledge (RAMP) and operational execution (GitHub repos). As the ecosystem grows across 6 modules and multiple versions, this separation enables scalability without confusion.
